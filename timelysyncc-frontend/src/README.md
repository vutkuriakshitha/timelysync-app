## Frontend Source Structure

This folder documents the frontend behavior in a cleaner source layout for review and maintenance.

Included here:

- `services/api.js`
- `services/authService.js`
- `context/AuthContext.js`
- `pages/Login.jsx`
- `pages/Signup.jsx`
- `pages/Register.jsx`
- `pages/VerifyEmail.jsx`
- `pages/ForgotPassword.jsx`
- `enhancements/source/*.js`

The files are organized to show the intended separation of concerns for API access, auth state management, user-facing auth pages, and the runtime enhancement layer that currently powers dashboard, create-task, and accountability improvements in the static frontend.
